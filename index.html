Research Focus

What I Work On

My work connects intelligent transportation systems and robotic navigation, with an emphasis on crosswalk safety, communication-aware decision making, and real-world mobile robot deployment.

SPaT/R2X-based Navigation

Communication-assisted crosswalk navigation for autonomous mobile robots using Signal Phase and Timing information.

ROS 2 / Nav2 Safety Systems

Integration of navigation, localization, signal awareness, and obstacle handling in a unified ROS 2-based safety framework.

VLM-based Intersection Understanding

Using vision-language models to interpret pedestrians, vehicles, occlusions, and scene context in intersection environments.

Selected Projects

Representative Research Projects

The projects below summarize the main technical direction of my current portfolio.

SPaT-Assisted AMR Crosswalk Navigation

A ROS 2-based framework integrating Nav2, SPaT/R2X messages, and a hybrid safety node for signal-aware AMR crossing.

ROS 2 · Nav2 · SPaT · R2X · AMR

Multi-Camera Pedestrian Trajectory Perception

A perception pipeline combining YOLO, DeepSORT, Re-ID, and Homography Transform for full pedestrian trajectory recovery.

YOLO · DeepSORT · Re-ID · Homography

CARLA-based Intersection Scenario Generation

A simulation pipeline for collecting intersection scenes, traffic actors, camera data, and metadata for VLM evaluation.

CARLA · Simulation · VLM Evaluation
